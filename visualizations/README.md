## 1. 相关性图 (Correlation Map) - `*_correlation_map.png`

### 图片内容
该可视化包含三张子图：
- 左图（Original Image）: 原始输入图像，红色框标注了示例框（exemplar boxes）的位置
- 中图（Similarity Map）: 相似度热力图，显示模型计算出的每个像素位置与示例的相似度
- 右图（Overlay Visualization）: 相似度图叠加在原始图像上的效果

### 含义解释
- 相似度图反映了模型在图像中寻找与示例相似区域的能力
- 颜色越亮（红色/黄色）表示该位置与示例的相似度越高，更可能是目标对象
- 颜色越暗（蓝色）表示相似度越低，更可能是背景区域
- 该图是模型的核心输出之一，用于后续的密度图生成

## 2. 动态权重 (Dynamic Weights) - `*_dynamic_weights.png`

### 图片内容
一个热力图，展示动态通道注意力权重：
- 横轴（Channel Dimension）: 特征通道维度
- 纵轴（Exemplar Index）: 不同示例的索引
- 颜色深浅: 表示注意力权重的大小（红色=高权重，蓝色=低权重）

### 含义解释
- 动态权重是 BMNet+ 模型的核心创新，用于自适应地选择不同通道特征
- 每个示例会生成一组权重，用于加权组合不同通道的特征
- 相同类别的示例通常会产生相似的权重模式（行与行之间相似）
- 不同类别的示例会关注不同的通道模式，实现类别自适应的特征选择

### 如何阅读
1. 观察每一行（每个示例）的权重分布模式
2. 如果多个示例的权重模式相似，说明它们属于同一类别
3. 权重高的通道（红色区域）表示该示例更关注这些特征通道

## 3. 密度图 (Density Map) - `*_density_map.png`

### 图片内容
该可视化包含两行三列共 6 张子图：

第一行（Ground Truth）:
- 左图: 原始图像
- 中图: 真实密度图（Ground Truth Density Map），显示每个位置的目标密度
- 右图: 真实密度图叠加在原始图像上

第二行（Prediction）:
- 左图: 原始图像
- 中图: 预测密度图（Predicted Density Map），模型预测的密度分布
- 右图: 预测密度图叠加在原始图像上

标题信息: 显示真实计数（GT Count）和预测计数（Pred Count），以及平均绝对误差（MAE）和相对误差百分比

### 含义解释
- 密度图是计数任务的核心输出，每个像素值表示该位置的目标密度
- 真实密度图是从标注数据生成的，通常使用高斯核在标注点周围生成密度分布
- 预测密度图是模型学习生成的，应该尽可能接近真实密度图
- 计数结果通过将密度图的所有像素值求和得到

### 如何阅读
1. 对比真实和预测密度图，观察它们的相似程度
2. 密度值高的区域（亮色）表示目标对象集中的位置
3. 叠加图可以帮助直观地看到目标在图像中的分布
4. 误差信息（MAE 和百分比）反映了模型的计数精度
5. 如果预测密度图与真实密度图越相似，说明模型性能越好
